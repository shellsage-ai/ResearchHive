# Safety System (Guidance + Environment Marking)

## Purpose
For physical-world domains, always label:
- recommended environment (desk / ventilated area / fume hood / pro lab)
- minimum PPE
- hazards + disposal notes
- references when available

## Policy
- Provide broad research knowledge and references.
- Avoid step-by-step dangerous procedural enablement.
- If a request crosses into dangerous detail: provide high-level overview + safer alternatives + references.
