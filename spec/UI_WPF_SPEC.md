# UI Spec (WPF .NET 8)

## Shell
- Sessions sidebar with search, filters, tags, status, last-report preview.
- Tabs: Overview, Research, Discovery Studio, Sources, Notebook, Jobs, Reports, Replay, Logs.
- Pack tabs: Materials Explorer, Programming Research+IP, Idea Fusion, Math Pad, Timeline.

## UX
- Async-safe, no UI thread blocking.
- Progressive disclosure to keep it clean.
- Citations always clickable; safety/IP badges visible where relevant.
