# Domain Capability Packs

## Goal
Each session uses a domain pack that defines:
- search lanes
- extraction and outputs
- scoring rubrics
- UI widgets
- safety/IP policy overlays

## Required packs (MVP)
- General Research
- History/Philosophy
- Math
- Maker/Materials
- Chemistry-Safe
- Programming Research + IP

## Pack selection UX
- New Session wizard: select pack (with auto-suggest option)
- Pack changes logged
